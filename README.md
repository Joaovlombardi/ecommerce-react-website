# ShopHub

A simple e-commerce application built with React to practice core front-end development concepts.

## Technologies

* React
* React Router DOM
* React Hook Form
* Context API
* LocalStorage

## Features

* User registration and login
* Authentication persistence
* Product listing
* Product details page
* Shopping cart management
* Global state management with Context API

## Concepts Applied

* Component-based architecture
* State management with React Hooks
* Context API
* Client-side routing
* Form handling and validation
* Data persistence with LocalStorage

## Getting Started

```bash
npm install
npm run dev
```

## Purpose

This project was developed for learning purposes, focusing on building a complete React application while practicing state management, routing, forms, and data persistence.
